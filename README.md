\# My Neural Network



\## 概要

C++で作成した基本的なニューラルネットワークライブラリです。



\## 特徴

* 標準ライブラリのみを使用しています。
* 逆伝播を自作しています。
* 行列演算を自作しています。
* SIMD命令を書いています。
* SIMD命令はコンパイル時に-mavx -mfmaを付けることで有効化できます。



\## 使い方

g++ *.cpp nn/*.cpp -Iexternal -o main

main

\## 注意
モデルの初期化が未熟なためLeakyReLU以外だと安定しないことがあります


\## 使用例

* 使用例はmain.cppの中にxorを学習させる場合を書いています。



\## 構成

* nnフォルダーの中にすべて入っています。



\### Activation

活性化関数とその微分を保持します。



\### Layer

重みやバイアスなど順伝播、逆伝播に必要な情報を保持します。



\### Matrix

基本的な行列です。



\### Trainer

モデルを保持し学習、推論、モデルの読み書きができます。
